Principais formatacoes:

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B(nao semantica)

Nesta frase temos um termo em Destaque ultilizando a tag STRONG(semantica)

Italico / Enfase

Nesta frase temos um termo em italico ultilizando a tag I(nao semantica)

Nesta fase temos um uma Enfase usando a tag EM(semantica)

Texto marcado

Podemos usar tambem um texto marcado ultilizando a tag MARK

Texto grande e pequeno

Estamos criando um texto grande e um pequeno nesse paragrafo!

Texto deletado

podemos indicar um texto deletado, para indicar que ele deve ser lido mas nao considerado

Texto inserido

podemos marcar um texto para dizer que ele foi adicionado depois

Texto sobreescrito

Para inserir coisas do tipo x20 + 3

Texto subescrito

Para inserir coisas do tipo H2O
